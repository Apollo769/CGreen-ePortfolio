Professional Self-Assessment

Academic Journey and Career Goals

Throughout the Computer Science program, I developed a foundation in software development, software engineering, databases, networking, and cybersecurity. While I entered the program with an interest in information technology, networking, and cybersecurity, my coursework helped me discover an interest in software development and software security. Through projects, coursework, and the capstone experience, I developed both technical and professional skills to pursue a career in technology. This ePortfolio reflects the growth of my knowledge, skills, and professional interests throughout the program and demonstrates my preparation for a career in software development and software security.

Communication and Collaboration

One of the most important lessons I learned through the program was the value of communication and collaboration. Through software design documents, technical reports, presentations, and project narratives, I strengthened my ability to explain technical concepts clearly and professionally. Collaboration was reinforced through instructor feedback, project reviews, and workplace experiences, which taught me how to evaluate feedback, communicate technical decisions, and work towards solutions that meet requirements.

Software Development

The program developed my understanding of software development principles, programming languages, and software engineering practices. Through coursework, I gained experience developing application in Java, C++, Python, JavaScript, HTML, and CSS while learning how different technologies can be applied to solve real-world problems. For example, full-stack web development using MEAN stack and the artifacts in this portfolio provided opportunities and experience designing, implementing, testing, and maintaining software systems.

Data Structures and Databases

In addition to software development, I gained experience working with data structures, algorithms, and databases. Through coursework and project development, I applied concepts such as searching, sorting, filtering, performance optimization, and data management while working with SQLite, MongoDB, and Firebase Firestore. These experiences strengthened my ability to evaluate technical trade-offs and reinforced the relationship between efficient algorithms, effective data organization, and overall software performance.

Security and Networking

Networking and cybersecurity coursework became some of the most influential areas of study throughout the program and further shaped my interest in security. I gained knowledge related to network design, firewalls, virtual local area networks (VLANS), access controls, authentication systems, compliance considerations, and secure software development practices. The capstone course provided an opportunity to apply these concepts through Firebase Authentication, Firestore security rules, input validation, and user-specific data access controls.

Artifact and Portfolio Summary

The artifacts included in this portfolio demonstrate the culmination of the knowledge and skills developed throughout the capstone and the program. First, my original application, A.I.M (Agile Inventory Management), establishes the foundation of the capstone project by providing an overview of the application’s original functionality, architecture, and limitations. The accompanying narratives for each enhancement and the code review video further document the analysis, planning, and implementation. Next, the software engineering and design, algorithms and data structures, and database enhancements demonstrate how the original application was transformed into a more maintainable, efficient, and secure system. Additionally, the user interface comparison illustrates the visual and functional improvements made through the capstone project.

Conclusion

Overall, the program has prepared me to pursue a career in software development and software security with a diverse technical skill set and a deeper understanding of how software, databases, networking, security, and emerging technologies interact within modern computing environments. More importantly, it taught me how to approach complex problems, evaluate technical trade-offs, communicate effectively, and continuously learn as technology evolves. The experiences and knowledge gained throughout the program have prepared me to continue developing my skills and pursue a career in software development and software security.
